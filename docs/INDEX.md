# Frontend Documentation

Dokumentasi frontend untuk System LLM.

## File-File Dokumentasi

- **README.md** - Panduan setup dan installation untuk development lokal
- **CLAUDE.md** - Development context, architecture, dan implementation details

## Struktur Folder Frontend

```
frontend/
├── app/                 # Next.js App Router dengan pages dan API routes
├── components/          # React components (admin, assistant-ui, ui, common)
├── hooks/              # Custom React hooks
├── lib/                # Utilities, services, types, hooks
├── store/              # Zustand state management
├── types/              # TypeScript type definitions
├── public/             # Static assets
├── docs/               # Documentation (folder ini)
└── Configuration files
```
